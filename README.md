# app

### Iniciando o projeto:
criar ambiente virtual:

py -3 -m venv venv

ativar ambiente virtual no Windows:

venv\Scripts\activate

instalar bibliotecas:

pip install -r requirements.txt

rodar a aplicação no Windows:

python wsgi.py
